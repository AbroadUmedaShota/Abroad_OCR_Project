# 00_PROJECT_OVERVIEW.md

## 1. プロジェクトの背景と目的

本プロジェクトは、機微情報を含むPDF文書を完全オフライン環境でOCR処理し、検索可能なPDFへ変換するアプリケーションの開発を目的としています。特に、日本語の手書き文字を含む混在文書において、高いOCR精度とバウンディングボックスの正確性を実現することを目指します。

## 2. プロジェクトの目標

- **完全オフライン動作**: ネットワーク接続なしでOCR処理を実行できること。
- **高精度OCR**: 日本語の手書き文字および印字文字に対応し、高い文字認識精度を達成すること。
- **検索可能PDF生成**: OCR結果を埋め込んだ検索可能なPDFを生成できること。
- **OCR結果の構造化出力**: OCR結果をCSV形式で出力し、後続処理での利用を可能にすること。

## 3. 主要機能 (Proof of Concept)

現在のProof of Concept (PoC) では、以下の機能が実装されています。

- **PDFから画像への変換**: 入力されたPDFファイルをページごとに画像ファイルに変換します。
- **PP-OCRv5によるOCR処理**: 変換された画像に対してPaddleOCR (PP-OCRv5) を用いて文字認識を行い、テキストとバウンディングボックス情報を抽出します。DBNet++による高精度なテキスト領域検出が統合されています。Ensemble VotingとKenLM補正の枠組みが導入されています。**LoRA微調整と精度レビューの枠組みが導入されています。**
- **OCR結果のCSV出力**: 抽出されたOCR結果を、ページ番号、ブロックID、座標、テキスト、信頼度を含むCSV形式で出力します。

## 4. 対象ユーザー

- 機微情報を扱う企業や組織の従業員
- オフライン環境での文書管理が必要なユーザー

## 5. スコープ

本PoCでは、PDFの画像変換、OCR処理、CSV出力に焦点を当てています。検索可能PDFの生成やGUIは今後の開発フェーズで実装予定です。
